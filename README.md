# 10 AI Portfolio Apps

A collection of 10 AI-powered applications built with React, TypeScript, and Vite.

## Apps
1. AI Code Reviewer
2. Document Chat
3. Image Generator
4. Voice Assistant
5. Code Explainer
6. Test Generator
7. API Integrator
8. Data Visualizer
9. Autonomous Agent
10. RAG Knowledge Base

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- @tanstack/react-query

## Development
```bash
npm install
npm run dev
```
